Universal Icons let you insert Icons easily Before and After Widgets Title. 

== Description ==
**Please drop me a message, if there is any problem in using the plugin.**

This plugin allows you to set Open Source Icons(Currently includes Font Awesome Icons) before or after widget title.

== Installation ==

How to install?


1. Upload `ts-icons-universal-free` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

Or
Install it via WordPress Plugins menu.

1. Add New Plugin
2. Search Universal Icons
